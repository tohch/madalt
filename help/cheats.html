Подсказка

После изменения сетевых настроек примените их командой: sudo systemctl restart networking

Примечание

Система ALT SP Workstation 11100-01 на дату 22.04.2026 имеет ядро 6.12

Внимание

После обновления ядра система может потребовать перезагрузки.

Важно

Всегда проверяйте контрольные суммы критических конфигурационных файлов после изменений!

Введение

#
user
root
>
auto eth0
" Основные настройки Vim
set tabstop=4
Установка пакетов

Скачайте скрипты и инструкцию:

wget -O ~/Загрузки/madalt.zip https://github.com/tohch/madalt/archive/refs/heads/main.zip
unzip -d ~/Загрузки -o ~/Загрузки/madalt.zip
mkdir -p ~/altlinux && yes | cp -rf ~/Загрузки/madalt-main/* ~/altlinux

Для скачивания установите ydiskarc, см. раздел Установка ydiskarc

Скачайте MS Office 2010

~/.local/bin/ydiskarc sync https://disk.yandex.ru/d/yF7r4A-K7-uPBw -o ~/altlinux/dist/office2010
